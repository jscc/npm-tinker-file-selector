NodeJs文件筛选器

npm install @tinker/file-selector

  • 快速筛选出你想要的文件列表

    只需要传入你想要筛选的目录和期望文件的后缀名即可

  • 自定义高级条件筛选

    允许你有更复杂的筛选条件，只需要传匹配文件的正则即可

  • // 安装
  • npm install @tinker/file-selector
  • // 引入
  • const fsr = require('@tinker/file-selector')
  • // 使用
  • console.log(fsr.select('./dir', 'js'))
  • // 使用-自定义筛选规则
  • console.log(fsr.advanceSelect('./dir', /\.js$/i))
